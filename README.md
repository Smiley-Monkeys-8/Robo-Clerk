# Robo-Clerk

## Getting Started

### Setup ENV

in `.env` file put the variables:

API_KEY: `your_api_key`
API_URL: https://hackathon-api.mlo.sehlat.io/

### Setup Poetry

#### Install Poetry
* install pipx: https://pipx.pypa.io/stable/installation/
* install poetry: `pipx install poetry`

#### Init Project
```
poetry shell
poetry install
```

### Run App

```
poetry run robo-clerk
```
